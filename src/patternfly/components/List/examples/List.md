## Documentation
### Overiew
Non-inline lists can be nested up to any level.

### Usage
| Class | Applied to | Outcome |
| -- | -- | -- |
| `.pf-c-list` | `<ul>, <ol>` | Initiates a list. **Required**  |
| `.pf-m-inline` | `.pf-c-list` |  Modifies for inline list style. |
