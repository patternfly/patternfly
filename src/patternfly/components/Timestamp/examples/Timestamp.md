## Examples

### Long format
```hbs
{{#> timestamp timestamp--attribute='datetime="2022-07-14T13:00"'}}
  Thursday, 14 July 2022, 1:00 PM EST
{{/timestamp}}
```

### Short format
```hbs
{{#> timestamp timestamp--attribute='datetime="2022-07-14T13:00"'}}
  14 July 2022, 1:00 PM EST
{{/timestamp}}
```

### Tiny format
```hbs
{{#> timestamp timestamp--attribute='datetime="2022-07-14T13:00"'}}
  14 Jul 2022, 1:00 PM EST
```
